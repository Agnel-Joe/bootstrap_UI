Scheduling

  • Scheduling Mode: Automatic, manual drag & drop and mixed mode scheduling. Scheduling considers team skills, working hours and required team size
  • Split of Task: Jobs can be split into multiple components with each component assigned to different teams, with the dependencies between components automatically managed
  • Schedule Type & View: Day, week and month views of schedule. Map view showing job locations and status

Mobile

  • IOS, Android, Web mobile app with biometric login and real-time integration with the backend scheduler. Supports journey planning, job execution and resource availability management.
  • Allows data capture of job attributes – including digital signatures, pictures, video, location data and standard form data fields.
  • Supports leave application workflow, including leave requests, approvals and rejections. Resource supervisors alerted when leave requests impact planned work, with approvals automatically triggering replanning if necessary.
